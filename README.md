# Аналитика пролонгаций: Автоматизация и BI-отчетность

Проект автоматизирует расчет коэффициентов удержания (пролонгаций) и визуализирует KPI отдела сопровождения.

## 📊 Результаты работы
* **Интерактивный дашборд:** [[ДАШБОРД В DATALENS](https://datalens.ru/hse6dh28j94m1)]
* **Автоматизированный отчет:** Сгенерированный Excel-файл с расчетами по каждому менеджеру.

![Дашборд](dashboard.png)

## 🏗 Архитектура решения
Вместо классического ведения тяжелых таблиц реализован современный аналитический пайплайн:
1. **Обработка (Python/Pandas):** Очистка сырых данных, агрегация и расчет пролонгаций за 1-й и 2-й месяцы.
2. **Представление (Excel):** Автоматическое сохранение результатов в читаемом бизнес-формате.
3. **Визуализация (DataLens):** Построение дашборда с фильтрами по менеджерам и динамикой KPI.

## 🛠 Технологический стек
* **Язык:** Python (Pandas, Openpyxl)
* **Инструменты:** Google Colab, GitHub
* **BI:** Yandex DataLens
